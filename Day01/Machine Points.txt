# * means we can accept more than one value
def add(*numbers):
   res = 0
   for num in numbers:
      res += num
   return res

What is deep learning?
----------------------
Neural networks

Inspired by how the huma processes information

using layers of neurons (input -> hidden layers -> output) to automatically extract some features

Learns Hierarchical Representation
   a. Early Layer -> simple features (edges in images)
   b. Deeper Layer -> Complex features (faces, object, meaning)
